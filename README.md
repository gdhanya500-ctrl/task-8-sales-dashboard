# Task 8 – Sales Dashboard

## Objective
Create an interactive dashboard showing sales performance by product, region, and month.

## Tools Used
Power BI

## Dataset
Superstore_Sales.csv

## Dashboard Features
- Sales trend over time
- Regional performance comparison
- Category distribution
- Interactive filter

## Insights
1. West region had highest sales.
2. Sales peaked during holiday months.
3. Technology category dominated revenue.

## Outcome
Learned how to design a clean business dashboard and extract insights visually.
